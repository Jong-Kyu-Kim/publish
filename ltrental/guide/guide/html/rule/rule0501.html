  • 테이블 - grid tb_row
    자격조건 홈페이지 회원가입
    무료 업그레이드 혜택 등급별 대여 횟수에 따라 차량 무료 업그레이드 서비스 제공 5번째, 15번째, 25번째, 35번째, 45번째, 55번째 (+10회 단위) 대여 시 소형 예약 차량 → 중형 업그레이드 중형 예약 차량 → 중대형 업그레이드
    • 대형, 수입차, SUV/RV, 승합 차량은 업그레이드 대상에서 제외됩니다.
    • 해당 차수만 이용 가능 (이월 없음)
      ※ 업그레이드 예약 후, 다른 대여건이 먼저 발생하면 차수가 변경될 수 있으니 유의하시기 바랍니다.
    • 업그레이드 서비스는 사전 예약제로 진행 (홈페이지 및 유선 예약 / 대여 6시간 전)됩니다.
      ※ 해당 지점 업그레이드 가용 차량이 있는 경우에 한해 서비스 이용 가능
      ※ 무료이용권 + 업그레이드 적용 시 실제 대여 할 업그레이드 후 차종과 소지하고 있는 무료이용권 차종이 동일할 때 사용 가능
    • 자세한 사항은 해당 지점으로 문의 바랍니다.

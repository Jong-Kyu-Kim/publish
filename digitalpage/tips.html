페이지 만들기 1

태그 적용하기

  • 1. 홈 화면 상단의 페이지 작성 영역을 선택하거나 하단의 페이지 생성 버튼을 눌러 새 페이지를 엽니다.

  • 2. 내용을 입력하세요. 제목을 정하거나 서식을 맞출 필요 없이 간단하게 내용을 완성하세요. 저장 버튼을 누르시면 페이지가 완성됩니다.

  • 3. 하단의 새 페이지 작성 버튼을 길게 터치하면 새 페이지 / 일정 /할 일을 바로 입력하여 페이지를 작성할 수 있습니다.

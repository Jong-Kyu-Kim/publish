페이지 만들기 [기본]

태그 적용하기

페이지에 태그를 적용해 보세요.

  1. 작성 중인 페이지의 태그 아이콘을 누르세요.

    태그 아이콘을 선택하면, 기존에 적용한 태그를 선택하거나 새롭게 입력할 수 있습니다.

  2. 페이지에 태그를 추가해두면 원하는 페이지 검색을 더욱 빨리 할 수 있습니다.

페이지에 배경색 지정하기

페이지에 배경색을 지정하여 기분이나 감정을 표현해 보세요.

  1. 작성 중인 페이지의 배경색 지정 아이콘을 누르세요.

  2. 19개의 색 중에 기분이나 감정을 표현할 수 있는 색을 지정해 보세요.

  3. 홈 화면 하단의 ‘Pages’를 누르세요 ‘TIMELINE’에서도 지정된 배경색을 확인할 수 있습니다.

할일 관리하기

개인 목표, 과제, 업무내용과 같은 크고 작은 일들을 놓치지 않도록 페이지에 바로 적어두세요.

  1. 작성 중인 페이지의 할 일 아이콘을 누르세요.

  2. 여러 개의 할 일을 한 번에 입력할 수 있습니다.

    할일 인라인태그가 적용되어 페이지에 입력됩니다.

    입력된 할 일을 선택하면 기한 설정을 할 수 있습니다.

    다양한 페이지에 기록한 할일은 하나의 리스트로 관리할 수 있습니다.

  3. 홈화면 하단의 검색 아이콘을 누르세요.

    검색 화면의 유형에서 '할일'을 누르고 우측 상단의 검색 버튼을 선택하면, 여러 페이지에 나눠 기록된 할일을 리스트로 보여줍니다.

  4. 검색 화면에서 유형을 '할일'로 선택한 후 특정 검색어를 입력하거나 태그를 추가 선택하면 빠르게 특정 할일을 찾아낼 수 있습니다.

  5. 할일이 마무리됐다면 목록에서 체크박스를 선택하여 완료 표시를 하거나, 해당 페이지로 이동해 체크박스를 선택하여 할 일을 완료 표시하세요.

사진 첨부해 메모하기

사진을 첨부해 페이지를 만들어 보세요.

  1. 페이지의 사진 아이콘을 누르세요.

  2. 새로 사진을 촬영하거나 장치에 있는 최근 사진을 선택해 첨부할 수 있습니다. 여러 장의 사진을 선택할 수도 있습니다.

  3. 페이지에 첨부된 사진을 좌우로 회전시키거나 디바이스에 저장할 수 있습니다.

    페이지 보기 상태에서 이미지를 터치하면 좌측 하단에 저장 아이콘을 확인할 수 있습니다.

    가운데 사진의 날짜 부분을 터치하면 사진의 정보도 확인할 수 있습니다.

    페이지 편집 상태에서 이미지를 터치하면 우측 하단에 사진 회전 아이콘을 확인할 수 있습니다.

음성 녹음으로 메모하기

음성 녹음으로 더 편리하게 메모해보세요.

  1. 페이지의 음성 아이콘을 누르세요.

  2. 녹음이 시작됩니다. 녹음을 마치려면 정지버튼을 눌러주세요. 녹음이 완료된 후 추가 버튼을 누르시면 녹음파일이 첨부됩니다.

  3. 어떤 내용을 녹음해뒀는지 기억할 수 있도록 간략한 메모를 남겨두세요.

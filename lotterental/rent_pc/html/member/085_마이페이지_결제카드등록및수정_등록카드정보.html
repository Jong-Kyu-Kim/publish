결제카드 등록이란?

신용카드로 결제할 때마다 카드 정보를 입력하지 않고 결제 시, 사용할 카드 정보를 한번만 등록해 놓으면 클릭 한번으로 간편하게 결제할 수 있는 서비스 입니다.

  • 등록 가능 카드 : 국민 / 비씨 / 삼성 / 현대 / 신한 / 롯데 / 하나SK / NH 농협 / 외한
  • 등록 불가 카드 : 해외 카드, 기프트 카드, 온라인 업종 사용 불가 카드
  • 등록 시, 사용된 정보는 저장되지 않고 이니시스에서 제공하는 키 값으로 결제가 진행됩니다.

홈 화면 활용하기

페이지 생성 도우미

  1. 페이지 생성 도우미로 페이지를 더 쉽고 빠르게 작성하세요. 휴대폰 활동 로그를 터치하는 것 만으로 페이지를 만들 수 있습니다.

  2. 홈 화면 왼쪽 상단의 페이지 생성 도우미 버튼을 눌러보세요. 스마트폰에서 생성된 지난 이틀간의 기록이 활동 로그로 보여집니다.

    • 새로 찍거나 저장한 사진
    • 추가나 수정한 연락처 정보
    • 새로운 일정이나 완료된 일정
    • 방문 장소

  3. 활동 로그를 바탕으로 페이지가 손쉽게 만들어집니다.

    생성된 페이지에 기억하고 싶은 내용을 간단한 메모로 추가하시면 더욱 생생한 기록이 완성됩니다.

추천 정보 확인하기

필요한 정보를 매번 검색하기 번거로우셨죠? DigitalPage를 열어보세요. 내 기록을 바탕으로 현재 상황에 필요한 정보를 찾기 전에 콕 집어 알려줍니다. 기억해야 하는 할 일, 일정은 물론, 지금 이 시점, 이 장소와 관련 있는 기록들을 추천해줍니다.

  1. ‘추천’ 항목을 더 보시려면 좌우로 밀어내기(swipe) 하세요.

  2. 페이지를 작성할 때 인라인 태그를 활용해보세요. 일정, 할 일, 장소를 포함한 페이지 정보가 꼭 필요한 상황에 맞춰 더욱 정교하게 추천됩니다.

  3. 최근에 사용한 페이지는 홈 화면의 하단에서 바로 확인하세요. 최근에 열어본 페이지 순서대로 노출됩니다.

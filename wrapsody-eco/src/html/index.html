외부 협업을 위한 새로운 접근 전략

이제는 안전하고 효율적인 외부 협업을 위한 새로운 접근이 필요한 시점입니다.

  • 데이터 중심의 협업 기능

    기존의 메일 시스템이나 EFSS에서 버전 충돌로 인해 반복해왔던 작업을 더는 하지 않아도 되기 때문에 불필요한 리소스들을 혁신적으로 절감할 수 있습니다.

  • 워크 그룹 단위의 이력 파악

    협업 시에 워크 그룹 단위로 업무 관리 및 이력 관리가 가능하며, 내부 사용자와 외부 협력 업체가 같이 그룹을 이루어 편리하게 관리할 수 있습니다.

  • 다양한 보안과 모니터링

    공유 문서를 PC에 저장하거나 클라우드 스토리지에 보관할 때 암호화될 뿐 아니라, 사용 중인 데이터까지 암호화하므로 안전하게 외부 협업을 진행할 수 있습니다.

  • 빠르고 생산적인 협업 환경

    위치와 무관하게 문서를 열람할 때 항상 최신 버전으로 동기화되어 중복 파일에 대한 확인 과정 없이 생산적으로 협업을 진행할 수 있습니다.

  • ROT 데이터 최소화

    협업 관련 이력이 하나의 워크 그룹으로 통합 관리되어 업무의 흐름을 한눈에 파악할 수 있고, 불필요하게 생성되는 데이터를 최소화 할 수 있습니다.

  • 보안 위협 최소화

    상황에 따라 전체 공유(Open Sharing), 제한적 열람(Trusted Viewing), 암호화(Encryption), DRM, 추적관리 등 문서 유출에 대한 걱정 없이 외부와 협업할 수 있습니다.

자주 묻는 질문

1개월 무료 평가판을 사용해보십시오. 언제든지 취소할 수 있습니다.

  • 무료 체험은 무엇인가요?

    제한된 환경에서 Wrapsody eCo 클라우드 서비스를 30일간 무료로 이용할 수 있는 서비스입니다. 신청하시면 담당자가 검토 후 승인해 드립니다. 30일 무료 체험 기간 내에 다른 요금제로 변경이 가능하며 변경 즉시 결제 후 이용 가능합니다.

  • Basic / Plus / Premium 요금제의 차이점은 무엇인가요?

    Basic/Plus 요금제의 경우 전체 디스크 사용량 및 사용자가 초대할 수 있는 외부 사용자 수에 제한이 있습니다. 별도로 원하시는 조건으로 가입을 원하시는 경우 Premium의 문의하기를 통해 문의하시면 파수 담당자가 원하시는 조건으로 사이트를 개설할 수 있도록 도와드립니다.

  • Basic 또는 Plus 요금제를 사용 중인 경우 Premium으로 업그레이드할 수 있나요?

    각 등급에서 상위 등급으로 업그레이드는 다음과 같이 진행됩니다. 매월 결제를 진행하는 경우, 요금제 변경 시 선불로 지불된 비용 이외에 추가로 발생하는 비용이 1회 청구되며, 변경 다음 달부터 변경된 요금이 청구됩니다. 연 1회 결제를 진행하신 고객의 경우 추가 비용이 1회에 합산되어 청구됩니다. 결제는 기본적으로 PayPal을 통해 이루어지며, Premium 요금제를 이용하시는 경우에는 파수 담당자에게 문의하시면 다른 결제 수단으로 변경이 가능합니다.

  • 매월 결제와 연 1회 결제의 차이점은 무엇인가요?

    매월 결제의 경우 선택한 요금제의 요금 매월 청구되며, 연 1회 결제는 선택한 요금제의 1년 동안의 사용 요금이 청구됩니다. 연 1회 결제 시 2개월 요금이 할인되어 총 10개월 요금만 청구됩니다. 서비스 기간 내에 구독을 중지하는 경우, 구독 취소를 신청한 달의 말일까지는 요금이 청구되며, 나머지 비용은 환급됩니다. 10개월 이상 사용한 경우에는 환급 비용이 없으니 참고하시기 바랍니다.

  • 사이트 개설 신청은 어떻게 하면 되나요?

    사이트 개설을 위해서는 먼저Wrapsody eCo Cloud 웹사이트에서 회원 가입을 하셔야 합니다. 회원 가입 이후 사용자에게 맞는 요금제를 선택하시고, 사이트 개설을 진행하시면 됩니다. 무료 체험의 경우 파수 담당자가 확인한 후에 사이트 개설이 진행되며, Basic과 Plus 요금제의 경우 별도의 확인 절차 없이 개설이 진행됩니다. Premium 요금제를 구독하는 경우, 원하는 조건에 맞추어 사이트가 개설될 수 있도록 파수 담당자가 도와 드립니다.

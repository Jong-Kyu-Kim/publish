페이지 쉽게 찾기

페이지 보기(시간/장소)

작성한 페이지들을 시간과 장소에 따라 분류된 것을 확인해보세요.

  1. 홈 화면 하단의 ‘Pages’ 아이콘을 선택하세요.

  2. ‘TIMELINE’에서는 시간의 흐름에 따라 작성된 페이지를 볼 수 있습니다.

  3. ‘PLACE’에서는 작성된 위치로 구분하여 어느 위치에서 얼마만큼의 페이지를 작성했는지를 볼 수 있습니다.

    페이지 개수로 구분된 영역을 선택하면 작성한 페이지의 리스트를 볼 수 있습니다.

    또한 상단의 날짜 검색 바를 선택하여 검색 날짜를 지정하면 지정된 날짜에 작성된 페이지를 위치별로 검색할 수 있습니다.

  4. ‘CALENDAR’에서는 날짜별로 구분된 페이지를 볼 수 있습니다. 작성된 페이지만이 아니라 모바일 장치에서 기본으로 제공하는 캘린더의 일정 또한 확인할 수 있습니다.

    좌우로 스크롤 하여 월단위로 이동이 가능합니다.

    어느 날짜에서든지 우측 상단의 '오늘' 버튼을 누르시면 오늘 날짜로 이동합니다.

페이지 검색

DigitalPage는 정교한 페이지 검색이 가능합니다.

  1. 1. 검색어를 넣고, 페이지 유형, 태그 정보와 같은 추가 정보를 선택해보세요. 페이지 검색이 빠르고, 쉬워집니다.

  2. 유형에 따라 페이지를 검색해보세요. 사진/음성/할일이 첨부되어 있을 경우 해당 유형으로 검색을 선택해 볼 수 있습니다.

    사진을 선택하고 검색 버튼을 누르면 사진 페이지만 모아서 볼 수 있고, 음성을 선택하면 음성녹음이 포함된 페이지를 검색할 수 있습니다.

  3. 할일을 관리하고 싶을 땐 유형을 '할일'로 선택하고 우측 상단의 검색 버튼을 누르세요.

    여러 페이지에 작성한 할일을 하나의 리스트로 보여줍니다. 특정 할일을 누르면 할일이 포함된 페이지가 보여집니다.

페이지 즐겨찾기

자주 확인해 봐야 하는 내용을 기록해 두었는데 찾기가 번거로우셨다면 즐겨찾기를 활용해 보세요.

  1. 즐겨찾기 하고 싶은 페이지를 선택하세요.

  2. 화면 우상단의 즐겨찾기 아이콘을 선택하세요.

  3. 홈 화면 즐겨찾기에서 확인할 수 있습니다. ‘>’ 화살표 버튼을 선택하여 목록으로도 확인할 수 있습니다. 또한 검색 화면의 ‘즐겨찾기’ 버튼을 눌러서 확인 할 수 있습니다.

  4. ‘TIMELINE’에서도 바로 즐겨찾기 하실 수 있습니다.

    홈 화면 하단의 ‘Pages’를 누르세요. ‘TIMELINE’에서 즐겨찾기 하고 싶은 목록을 길게 누르세요. 즐겨찾기 아이콘을 선택하여 즐겨찾기에 추가할 수 있습니다.

페이지맵

기록을 일일이 분류하거나, 필요한 기록을 일일이 찾아보시나요? 연관페이지로 묶어두면 관련된 내용끼지 서로 연결되어 원하는 정보를 쉽게 찾아 활용할 수 있습니다.

  1. 페이지 상단 오른쪽의 '페이지 맵' 버튼을 누르세요. 페이지의 연관페이지를 확인할 수 있습니다.

  2. 연관페이지의 썸네일을 누르세요. ‘연관/추천보기’를 선택하면 그 페이지의 연관 페이지들까지 연결을 확대해 보여줍니다.

  3. 내가 작성한 페이지 전체의 연관 관계를 시각화해서 볼 수 있습니다.

    홈화면 하단의 ‘Search’ 아이콘을 누르세요. 화면 상단의 '페이지맵' 아이콘을 누르세요. 페이지의 연결관계가 별자리와 같이 보여지며, 작성한 전체 페이지 개수와 페이지 간 연결 개수를 확인할 수 있습니다.

    각각의 페이지가 점으로 표시되어 연결되며 최근 열람한 페이지일 수록 밝게 표시됩니다.

  4. 엄지와 검지 두 손가락을 벌려(핀치) 확대하면 페이지 썸네일이 보여집니다.

    페이지맵의 연관페이지 모음을 확인하면 주로 어떤 내용을 주제로 기록을 남기는지, 어떤 기록들이 연결되는지를 확인할 수 있습니다.

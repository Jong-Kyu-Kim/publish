일정 관리하기

DigitalPage를 활용해 일정을 관리해보세요. 일정만 관리할 수 있는 캘린더 앱에 비해 DigitalPage는 달력을 통해 날짜별 기록과 할일, 스마트폰 기본 캘린더에 저장한 일정까지 모두 확인할 수 있어 모든 기록을 통합 관리할 수 있습니다.

  1. 작성 중인 페이지의 우측아이콘을 누르세요.

  2. 일정 아이콘을 선택하면, 일정을 입력할 수 있습니다. 기간을 선택하고 반복여부를 선택하세요.

  3. 일정 인라인태그가 적용되어 페이지에 입력됩니다.

  4. 일정을 관리하거나 기본 캘린더 일정을 DigitalPage의 일정으로 입력해 관리하려면 달력 보기 화면으로 이동하세요.

    화면 하단 탭의 ‘Pages’ 화면에서 상단의 ‘CALENDAR’를 선택하면 달력 보기 화면이 제공됩니다.

  5. 기기의 기본 캘린더 일정을 DigitalPage의 일정으로 입력해 관리하려면 해당 일정을 누르세요.

    DigitalPage 일정 입력 창에 일정 정보가 자동으로 입력되어 '저장' 버튼만 누르시면 됩니다.

페이지에 장소 넣기

페이지에 장소나 위치를 입력할 때는 장소 인라인 태그를 적용해보세요. 별도의 지도앱을 켤 필요 없이 페이지 내에서 해당 위치를 지도로 연결해 보여줍니다.

  1. 작성 중인 페이지의 우측아이콘을 누르세요. 장소 아이콘을 선택하면, 장소을 입력할 수 있습니다.

  2. 검색창에 장소명이나 주소를 입력해 해당 장소를 찾아보세요. 사용자가 이미 적용한 인라인 태그가 있다면 하단의 목록 위쪽에 노출되며, 현재 위치를 기준으로 추천 장소가 노출됩니다.

    장소를 선택하고 완료를 누르세요.

  3. 페이지 내에 인라인태그가 적용된 장소 정보가 입력됩니다.

  4. 페이지 저장을 누르고 난 후 인라인태그가 적용된 장소를 누르면 해당 장소의 지도가 바로 표시됩니다.

페이지에 날짜 넣기

페이지에 날짜와 시간을 입력해보세요.

  1. 작성 중인 페이지의 우측 아이콘을 누르세요. 날짜 아이콘을 선택하면, 날짜와 시간을 입력할 수 있습니다.

  2. 날짜를 선택하고, 시간을 추가하려면 시간 설정 버튼을 on으로 전환하세요. 시간 설정까지 마쳤다면 완료 버튼을 누르세요.

  3. 페이지 내에 입력한 날짜와 시간이 표시됩니다.

    페이지 내용에 날짜 인라인태그를 적용해두면 해당 날짜에 홈 화면 추천 콘텐츠로 노출되어 필요한 날짜에 페이지를 확인할 수 있습니다.

페이지에 연락처 넣기

  1. 작성 중인 페이지의 우측 아이콘을 누르세요. 연락처 아이콘을 선택하면, 기존에 추가된 연락처 목록이 나타납니다.

  2. 연락처에 이름이나 전화번호를 입력하면, 모바일 장치의 연락처에 있는 내용도 목록에 나타납니다.

  3. 목록에 있는 연락처를 선택하거나 새로운 이름 또는 전화번호를 입력 후 완료 버튼을 누르세요.

  4. 페이지 보기 상태에서 연락처를 선택하면, 전화를 걸거나 메시지를 보낼 수 있습니다.

페이지 알림

페이지에 알림을 적용해 보세요.

  1. 작성 중인 페이지의 우측 아이콘을 누르세요.

  2. 알림 아이콘을 선택해서 알림을 원하는 날짜와 시간을 설정하세요.

    완료 버튼을 선택하면 페이지 내에 알림 정보가 노출됩니다. 한 페이지 내에는 여러 개의 알림도 설정할 수 있습니다.

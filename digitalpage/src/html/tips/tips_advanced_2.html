페이지 응용하기

페이지 보내기

  1. 페이지 보기 화면에서 우측 상단의 버튼을 누르세요.

    페이지 보내기 버튼을 누르면 받는 사람 입력 창이 뜹니다.

  2. 받는 사람의 이메일 주소를 입력하세요.

    이메일을 계속 추가하여 여러 명에게 보낼 수 있습니다.

  3. 만약 보낸 페이지를 상대방이 확인했는지 궁금하다면 "읽음 확인" 옵션을 켜고 보내세요.

    상대방이 페이지를 읽으면, 읽음 확인 알림을 받을 수 있습니다.

  4. 페이지 정보에서 누구에게 보낸 페이지인지, 누구에게 받은 페이지인지 확인할 수 있습니다.

페이지 합치기

페이지를 선택해서 합쳐보세요.

  1. 페이지 보기 화면에서 우측 상단의 버튼을 누르세요. 페이지 합치기 버튼을 선택하면 전체 페이지 목록이 나타납니다.

  2. 검색하거나 페이지 목록에서 병합할 페이지를 찾아서 페이지 선택 버튼을 누르세요.

    병합할 페이지는 1개만 선택됩니다.

    합치기 버튼을 누르시면 페이지가 병합됩니다. 선택된 페이지는 이전 페이지 하단에 추가됩니다.

페이지 연결하기

페이지를 관련 있는 다른 페이지와 연결해보세요.

  1. 페이지 작성을 완료하면 관련된 기록들이 연관페이지 추천으로 보여져 손쉬운 페이지 연결이 가능합니다.

    '연결'을 눌러 페이지를 연결하세요.

  2. 추가로 페이지를 연결하거나 수동으로 페이지를 연결하고 싶다면 페이지 우측 상단의 '페이지 맵' 아이콘을 누르세요.

    해당 페이지에 대한 연관 페이지와 추천 페이지의 연결을 페이지 썸네일로 시각화해서 보여줍니다.

    1. 추천 페이지 연결

      연관페이지로 추천하는 페이지의 썸네일은 보라색 표시되며, 해당 내용을 누르면 연관 페이지로 연결할 수 있습니다. 연관페이지 추천은 최대 2개까지 제공됩니다.

    2. 페이지 수동 연결

      페이지를 수동으로 연결할 수 있습니다. 페이지 썸네일을 누르세요. 연결할 페이지를 새로 작성하거나 검색을 통해 연결할 수 있습니다.

    3. 페이지 연결 삭제

      페이지의 연결을 삭제하려면 연관페이지의 썸네일을 누르세요. 연결 끊기를 눌러 연결을 끊을 수 있습니다.

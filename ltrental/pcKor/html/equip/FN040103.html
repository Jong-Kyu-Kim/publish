측정기 렌탈

측정기 렌탈이란?

  • 유선 및 무선 통신 분야 장비 및 시스템에 대한 전자, 전기, 광학적 특성 신뢰성 테스트용 측정 장비를 다양하게 보유하고 있어 고객이 원하는 기간만큼 렌탈해 드립니다. 또한 고객이 합리적인 가격으로 측정기를 구매할 수 있도록 신규/중고장비 구매 부터 렌탈 후 양도 가능한 서비스까지 제공하고있습니다.
문의(고객 센터) 대표전화 1577-5100
  • 일반렌탈 상품에 대한 궁금하신 사항 있으시면 무엇이든 문의하세요. 궁금한 사항을 바로 해결해 드립니다.

상품 종류

국내 최대 규모의 유선, 무선 측정기를 보유 및 운용

  • 무선통신(RF, 광통신, 무선단말기)분야 약 7,000대
  • 유선통신(전기, 전송)분야 약 5,000대
무선/유선/광통신 관련 측정 기류
  • Spectrum Analyzer, Signal Generator
  • Network Analyzer, Mobile Test Set
  • 광선로 측정기, 패킷분석기
  • 웹 보안관련 측정기, 인터넷 측정기
  • 기타 측정기
설비류
  • 대형 Room Chamber(5m*3m*2m)
  • 중형 Room Chamber(3m*3m*2m)
  • 소형 Chamber(1m*1m*1.5m)
  • RF Shield Room
측정 자동화 프로그램 지원
  • 측정기 사용의 편의를 위해 PC프로그램을 통하여 측정기를 제어할 수 있는 Auto Test System i-man을 제공합니다.
  • 복잡한 조작 과정없이 i-man으로 측정기를 제어할 수 있도록 최적의 환경을 구축해 드립니다.

이용 조건

기준 렌탈료 1개월 사용료 (월 기본 렌탈료)
할인 적용
  • 2개월 이상 : 장기 할인 적용
  • 1개월 미만 : 단기 할인 적용 (1주일 : 기준 렌탈료 x 30%, 2주일 : 기준 렌탈료 x 60%, 3주일 : 기준 렌탈료 x 80%)
  • 12개월 이상 : 별도 협의 후 렌탈료 산정
납부 방법
  • 월 선불 납부
  • 최초 1회분 렌탈료는 물건 인수 시 납입, 2회분부터는 무통장 납입
  • 소모품 비용은 별도 부담

이용 절차

  • 01. 렌탈 이용자가 품목 산정 후 렌탈 신청

    (전화, Fax, 이메일등)

  • 02. 렌탈계약 체결/해당기기 고객인도

    (계약서 발송 및 계약서 작성)

  • 03. 기기 선정 및 해당 기기 고객인도 / 제품이상

    유무 및 부속품 확인 후 수령증 접수

  • 04. 렌탈 이용자 렌탈료 납부개시
  • 05. 렌탈 중인 물품에 대한 유지보수

    (고장 발생 시, 동급 장비로 대체)

  • 06. 사용 후 기기 반납

    (검, 교정 및 부속 확인)

기간 연장

  • 연장방법: 렌탈기간 종료일로부터 1주일 전에 전화, Fax, 이메일 등으로 신청합니다.
  • 렌탈료: 연장기간 동안은 총 사용기간 (기 사용기간 + 연장기간)에 해당하는 렌탈요율 적용합니다.

중도 해지

  • 해지방법 : 해지 예정일로 부터 1주일 전에 전화, Fax, 이메일 등으로 신청합니다.
  • 렌탈료 : 해약시점에서 실 사용기간에 상당하는 요금을 정산하여 그 차액만 납부합니다.

기타

페이지 작성일 변경하기

여행을 다녀온 뒤 여행 기록을 남기려는데 그 시간의 감동을 그대로 남기고 싶다면 페이지 작성일을 변경해 보세요.

  1. 페이지 편집 상태에서 화면 상단 중간의 날짜 부분을 선택하세요.

  2. 변경하고 싶은 날짜 및 시간을 선택하세요. 내용을 작성 후에 완료 버튼을 누르세요.

    ‘TIMELINE’에서 변경된 작성일로 페이지가 보여지는 것을 확인할 수 있습니다.

웹 링크 활용하기

웹 서핑 중에 내용 자체를 기록하여 남겨두고 싶을 때는 웹 링크 기능을 활용해 보세요.

  1. 웹 서핑한 페이지의 URL을 복사하세요.

  2. 페이지 작성 버튼을 누르세요.

  3. 페이지에 복사한 URL을 붙여넣기 하고 Enter 키를 누르세요. 해당 URL의 Summary가 나타나게 됩니다.

  4. 페이지 보기 상태에서 해당 URL이나 Web Summary 를 선택하면 해당 URL로 이동하여 기록된 내용을 보실 수 있습니다.
